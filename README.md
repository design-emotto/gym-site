# IRON NEST GYM 公式サイト（1ページ）

架空のジム **IRON NEST GYM** の1ページ公式サイトです。

## ファイル構成

- `index.html` : セマンティックHTMLで構成した1ページ
- `styles.css` : 黒×白＋ゴールドアクセントのスタイル
- `README.md` : このドキュメント

## 実装内容

以下のセクションを実装しています。

- Hero（キャッチコピー + CTA 2つ）
- 特徴
- 料金プラン
- トレーナー紹介
- お客様の声
- アクセス
- お問い合わせ

## デザイン方針

- カラー: 黒 (`#0a0a0a`) × 白 (`#f4f4f4`) + アクセント（ゴールド `#c9a227`）
- 高級感: 広めの余白、細い境界線、タイポグラフィ強調
- モバイル対応: レスポンシブ（960px / 700pxでレイアウト調整）

## アクセシビリティ対応

- 適切な見出し階層（`h1`〜`h3`）
- セマンティック要素（`header`, `main`, `section`, `nav`, `footer`, `form`）
- スキップリンク
- フォーカス可視化（`focus-visible`）
- フォームの `label` と入力欄の関連付け
- コントラストを意識した配色

## 使い方

1. `index.html` をブラウザで開く
2. 内容変更時は `index.html` を編集
3. 配色や余白、レスポンシブ調整は `styles.css` を編集

## 補足

- 住所や料金、トレーナー情報はダミーです。
- 地図とSNSリンクはダミーリンク (`#`) を設定しています。
